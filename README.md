# 🌐 Personal Portfolio Website

A responsive, multi-page personal portfolio website built using **HTML, CSS, and JavaScript** to showcase profile, skills, projects, and experience.

🔗 **Repository:** [a-chitransh/Website](https://github.com/a-chitransh/Website)

---

## 📌 Project Overview

This portfolio is designed as a clean and structured personal website with dedicated pages for different sections of your professional profile.

### Included pages:
- **Home** (`index.html`)
- **About** (`about.html`)
- **Skills** (`skills.html`)
- **Projects** (`projects.html`)
- **Experience** (`experience.html`)

---

## ✨ Features

- Multi-page layout with clear section separation
- Responsive design for desktop and mobile
- Reusable styling via a shared CSS file
- Interactive behavior using JavaScript
- Easy to customize and extend

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/a-chitransh/Website.git
cd Website
```

### 2. Run locally

You can open `index.html` directly in your browser.

For better development experience, run a local server:

```bash
python -m http.server 5500
```

Then open: `http://localhost:5500`

## 🌍 Deployment

This is a static website, so you can deploy it easily on:

- **GitHub Pages**
- **Netlify**
- **Vercel**

### GitHub Pages (quick steps)

1. Open repository **Settings**
2. Go to **Pages**
3. Set source to your main branch root (`/`)
4. Save and wait for deployment
