## 🛠️ Technology Stack
The site is built entirely with vanilla web technologies to ensure lightning-fast performance and an elegant, minimalist aesthetic. No heavy frameworks were used.
* **HTML5:** Semantic structuring
* **CSS3:** Custom styling, smooth animations, and fully responsive layout (Mobile-first design)
* **JavaScript:** DOM manipulation, Intersection Observer for scroll animations, and interactive navigation elements.

## 🚀 Features
* **Modern & Minimalist aesthetic:** Black & white design with clean typography and blue accent highlights.
* **Smooth Scrolling & Fade-ins:** Utilizes JavaScript Intersection Observers for elegant, non-intrusive scroll animations.
* **Fully Responsive:** Adapts seamlessly to all screen sizes (desktop, tablet, mobile).
* **Organized Categorization:** Specific sections covering my biography, technical skills, core competencies, timeline of professional experience, and detailed project case studies.

## 👩‍💻 Setup for Local Development
Since this is a static site, running it locally is incredibly simple. You do not need to install `npm` or configure a complex backend environment.

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   ```
2. Navigate to the project directory:
   ```bash
   cd your-repo-name
   ```
3. Open `index.html` in your favorite web browser. 
   *(Alternatively, use an extension like VSCode Live Server or a simple Python http server: `python -m http.server 8000` to serve the files locally).*
