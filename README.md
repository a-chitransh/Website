## 🚀 Getting Started

### 1) Clone the repository

```bash
git clone https://github.com/a-chitransh/Website.git
cd Website
```

### 2) Run locally

Since this is a static website, you can run it by simply opening `index.html` in your browser.

For better development workflow, use a local server:

- VS Code + **Live Server** extension, or
- Python simple server:

```bash
python -m http.server 5500
```

Then open: `http://localhost:5500`

## 🎨 Customization Guide

You can personalize this portfolio quickly:

### Update personal info
- Edit headings, intro text, and bio in `index.html`.

### Update styles
- Modify colors, spacing, typography, and layout in your CSS files.

### Add projects
- Duplicate project card blocks in HTML and update:
  - project title
  - description
  - tech stack
  - links (GitHub/demo)

### Update contact/social links
- Replace placeholder links with your:
  - GitHub
  - LinkedIn
  - Email
  - Twitter/X (optional)

## 🌍 Deployment

You can deploy this project on any static hosting platform.

### Option A: GitHub Pages
1. Go to your repository on GitHub.
2. Open **Settings → Pages**.
3. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main` (or your default branch), folder `/root`
4. Save and wait for deployment.

### Option B: Netlify
- Drag and drop the project folder in Netlify dashboard  
**or**
- Connect GitHub repo for continuous deployment.

### Option C: Vercel
- Import repository from GitHub and deploy with default static settings.
